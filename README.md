# CMIsomer

Party registration and ticketing app built with Next.js + Supabase.

## Quick Start

1. Install: `pnpm install`
2. Configure `.env.local`:
   - `NEXT_PUBLIC_SUPABASE_URL`
   - `NEXT_PUBLIC_SUPABASE_ANON_KEY`
3. Run: `pnpm dev`

## Scripts

- `pnpm dev`
- `pnpm lint`
- `pnpm build`
- `pnpm start`
